## BuzzBooster SDK for React Native

* [개발 가이드](https://buzzvil.atlassian.net/wiki/spaces/BDG/pages/2953019933/BuzzBooster+React+Native+SDK)

## Quick Start
```sh
npm install .
cd ios
pod install

npx react-native run-android
npx react-native run-ios
```
